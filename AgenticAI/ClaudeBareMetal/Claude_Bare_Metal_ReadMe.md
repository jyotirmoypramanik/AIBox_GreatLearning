# Claude Bare Metal

A minimal implementation of Claude AI assistant for bare metal deployment.

## Features

- Lightweight footprint
- No external dependencies
- Direct hardware access
- Fast initialization

## Installation

```bash
# Build from source
make build

# Deploy to bare metal
make deploy
```

## Usage

```bash
./claude-bare-metal --help
```

## Configuration

Edit `config.yaml` to customize settings.
